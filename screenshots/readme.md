# Screenshots

This directory contains screenshot evidence collected during the Wazuh SIEM lab deployment and testing process.

## Naming Convention
Screenshots are named numerically to reflect the chronological order of implementation and validation.

## Contents
- **01_dashboard_access.png** – Wazuh Manager dashboard accessibility verification  
- **02_agents_connected.png** – Linux and Windows agents successfully connected  
- **03_fim_alert.png** – File Integrity Monitoring alert  
- **04_privilege_escalation_alert.png** – Privilege escalation detection  
- **05_account_creation_alert.png** – User account creation monitoring  

Each screenshot corresponds to a documented configuration or detection step in the repository.
