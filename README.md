# Wazuh SIEM Lab

## Overview
This repository demonstrates a production-style Wazuh SIEM deployment designed to showcase SOC analyst and blue team capabilities. The lab focuses on centralized security monitoring, detection engineering, and incident investigation across Linux and Windows endpoints.

## Skills Demonstrated
- SIEM deployment and architecture design
- Endpoint security monitoring (Linux & Windows)
- Log analysis and alert triage
- Detection engineering fundamentals
- Incident investigation methodology

## Lab Architecture
- Wazuh Server (Ubuntu)
- Linux Endpoint (Ubuntu Agent)
- Windows Endpoint (Windows 10 Agent)

## Security Use Cases (Planned)
- Privilege escalation detection
- File Integrity Monitoring (FIM)
- Unauthorized user creation detection
